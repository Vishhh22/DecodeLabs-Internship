🔬 DecodeLabs Sensor Simulator

Real-time IoT Sensor Data Collection & Analysis

📋 Project Overview

A comprehensive sensor data simulator that collects, processes, and logs real-time sensor data. Built as part of the DecodeLabs Internship program to demonstrate IoT concepts, data handling, and sensor simulation.

🌡️ Temperature Sensor 💧 Humidity Sensor 🚨 Motion Sensor 📊 Data Logging 📈 Statistics

✨ Key Features

  • Multi-Sensor Support: Temperature, Humidity, and Motion sensors
  • Realistic Simulation: Natural drift and variation in readings
  • Real-time Display: Live sensor readings with formatted output
  • Dual Data Export: JSON and CSV formats for analysis
  • Statistical Analysis: Min, max, and average calculations
  • No Dependencies: Uses only Python standard library

🚀 Getting Started

Prerequisites

  • Python 3.7 or higher
  • No external dependencies required

Installation & Running

# Clone the repository
git clone https://github.com/Vishhh22/DecodeLabs-Internship.git
cd DecodeLabs-Internship

# Run the simulator (default: 30 seconds)
python sensor_simulator.py

# View sample output
python sample_output.py

Customization

from sensor_simulator import SensorSimulator

# Create simulator instance
simulator = SensorSimulator()

# Run with custom parameters
# duration_seconds: how long to run (default: 30)
# interval: seconds between readings (default: 2.0)
simulator.run_simulation(duration_seconds=60, interval=1.0)

# View statistics
simulator.get_statistics()
